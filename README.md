## Getting Started

### Cloning the Project:

git clone <https://github.com/delvalj/API-Rick_and_morty>

## Installing

###  How to Install it

Open the terminal and run:

- npm install

## Starting

### Open the project in LocalHost:

In the terminal:

- npm run serve

## Deployment

In order to deploy the project i used <https://app.netlify.com/>
